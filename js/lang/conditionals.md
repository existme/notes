# Conditionals

## If...Then...Else
    
``` js
if (a == 2) {
    // do something
}
else if (a == 10 || a == 11) {
    // do another thing
}
else if (a == 42) {
    // do yet another thing
}
else {
    // fallback to here
}
```

## Switch..Case
``` js
switch (a) {
    case 2:
        // do something
    break;
    case 10:
    case 11:
        // do another thing
    break;
    case 42:
        // do yet another thing
    break;
    default:
        // fallback to here
}
```

## Ternary operator
``` js
var a = 42;
var b = (a > 41) ? "hello" : "world";
```
