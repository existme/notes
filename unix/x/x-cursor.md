``` sh
gsettings get org.gnome.desktop.interface cursor-size
gsettings get org.gnome.desktop.interface cursor-theme

gsettings set org.gnome.desktop.interface cursor-size 64
```

1. 24: Default
1. 32: Medium
1. 48: Large
1. 64: Larger
